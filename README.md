# Retail Sales Performance Dashboard

## Objective
Analyze 25K+ retail orders to identify sales trends, 
regional performance, and product insights.

## Tools Used
SQL Server | Power BI | DAX

## Key Insights
- Analyzed 25K+ orders and ₹12.6M in total sales revenue
- Built KPI metrics for Total Sales, Profit, Orders, and Customers
- Identified regional and product-level performance trends
